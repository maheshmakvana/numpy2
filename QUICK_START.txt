================================================================================
                       PROBLEM SOLVED
================================================================================

BEFORE (without numpy2):
  >>> import numpy as np, json
  >>> arr = np.array([1, 2, 3], dtype=np.int64)
  >>> json.dumps(arr)
  TypeError: Object of type int64 is not JSON serializable

AFTER (with numpy2):
  >>> import numpy2 as np2
  >>> arr = np.array([1, 2, 3], dtype=np.int64)
  >>> np2.to_json(arr)
  '[1, 2, 3]'
